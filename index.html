How it works
Four things that make it different.

Not prompts. Not filters. Runtime structures that change what an agent can and cannot do — by design.

🚧

Instruction gate

Every action is tagged with its source — you, the agent, or external content. Web content can never trigger tool calls. Ever.

🔏

Signed identity

The agent's identity is cryptographically signed. Tamper with it and the agent refuses to start. No silent rewrites.

🙋

Caution layer

When the agent detects it's making too many mistakes, it pauses and asks before proceeding. Human oversight built in.

🔒

Encrypted memory

Everything stored is AES-256-GCM encrypted locally. API keys live in process memory only — cleared on shutdown.

Get started
Running in 5 minutes.

No account. No cloud. Bring your own API key — it never leaves your machine.

    Clone and install

    Clone the repo and install dependencies.

    git clone github.com/thecolourfoundation/Color && cd Color && npm install ⧉ Copy
  • 2

    Add your API key

    Works with Anthropic, OpenAI, Groq, and Ollama. Your key is never written to disk.

    export ANTHROPIC_API_KEY=sk-ant-... ⧉ Copy

    Run the demo

    Watch Colors block a live injection attack.
